# Contact Management Challenge

## Setup
```bash
# Option A: One-step setup
./setup.sh

# Option B: Manual setup
npm run install:all
npm run prisma:generate
```

## Development
```bash
# Terminal 1
npm run dev:server

# Terminal 2
npm run dev:client
```

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:3001`
